# TaubenAbwehrSystem
Dies ist die Grundlage eines TaubenABwehr systems das über Bilderkennung die Taben erkennen soll und gezielt element ansteuren kann umd die Taube von ihrer aktuellen Position geziet zu verscheuchen.

## Komponenten
Es soll eine Blender Pipeline entstehen die aus CAD Daten bilder generieren kann um damit eine KI trainieren zu können.
